# 샐러리아 침산점 — 영수증 양식

> 용지: 58mm 열전사 / 1배 폭 기준 42자  
> 정렬: 좌측 정렬 기본. 헤더만 가운데 정렬.  
> 주문번호·이용방법 값은 **볼드**.

---

## ① 주방용 영수증

```
          [주방용]
  샐러리아 침산점 - 선결제 영수증
------------------------------------------
주문번호 : 1301
주문일시 : 2026/07/01 12:02
------------------------------------------
이용방법 : 배달
주문자   : 김경민
전화번호 : 010-1234-5678
------------------------------------------
배달주소 : 대구 북구 침산동 123-45 2층
가게요청 : 수저·포크 X
배달요청 : 문 앞에 놓아주세요
------------------------------------------
메뉴명                        수량
- - - - - - - - - - - - - - - - - -
육샐 도시락                      1
  > 저칼로리 드레싱
  > 후레쉬버 추가
아이스 아메리카노 (상가사장님       2
  가격)
------------------------------------------
고객요청 : 없음
```

### 주방용 — 블록 설명

| 블록 | 조건 | 내용 |
|------|------|------|
| 헤더 | 항상 | `[주방용]` (가운데 볼드) + 가게명 (가운데) |
| 주문 정보 | 항상 | 주문번호, 주문일시 |
| 고객 정보 | 항상 | 이용방법 (볼드), 주문자, 전화번호 |
| 배달 정보 | 배달일 때만 | 배달주소, 가게요청, 배달요청 |
| 메뉴 목록 | 항상 | 메뉴명 + 수량 2열. 긴 이름은 자동 줄바꿈. 옵션은 들여쓰기 `>` |
| 고객요청 | 항상 | 없으면 "없음" |

```

------------------------------------------
주문번호 : 1301

------------------------------------------
이용방법 : 배달
주문자   : 김경민
전화번호 : 010-1234-5678
------------------------------------------
배달주소 : 대구 북구 침산동 123-45 2층
가게요청 : 수저·포크 X
배달요청 : 문 앞에 놓아주세요
------------------------------------------
메뉴명                    수량    가격
- - - - - - - - - - - - - - - - - - - -
육샐 도시락                  1   9,400원
  > 저칼로리 드레싱
  > 후레쉬버 추가
아이스 아메리카노              2   6,000원
  (상가사장님 가격)
------------------------------------------
메뉴 소계                       15,400원
배달료                           3,500원
------------------------------------------
합  계                          18,900원
------------------------------------------

```

잔액이 음수일 때:
```
주문후 잔액                     -18,900원
※ 잔액 부족 - 다음 충전 시 정산
```

### 고객용 — 블록 설명

| 블록 | 조건 | 내용 |
|------|------|------|
| 헤더 | 항상 | 가게명 (가운데) |
| 주문 정보 | 항상 | 주문번호, 주문일시 |
| 고객 정보 | 항상 | 이용방법 (볼드), 주문자, 전화번호 |
| 배달 정보 | 배달일 때만 | 배달주소, 가게요청, 배달요청 |
| 메뉴 목록 | 항상 | 메뉴명 + 수량 + 가격 3열. 긴 이름은 줄바꿈. |
| 금액 요약 | 항상 | 메뉴소계, 배달료(배달 시), 합계 |
| 잔액 | 항상 | 주문전/후 잔액. 음수 시 경고 문구 |

## 열 너비 기준 (42자, 1배 폭)

### 메뉴 목록 — 주방용 (2열)

```
[메뉴명 ~38자 자동 줄바꿈][수량 우측 4자]
```

### 메뉴 목록 — 고객용 (3열)

```
[메뉴명 ~26자 자동 줄바꿈][수량 4자][가격 우측 12자]
```

수량과 가격의 열 너비는 고정. 메뉴명이 길면 다음 줄로 이어짐:

```
아이스 아메리카노              2   6,000원
  (상가사장님 가격)
```

## 폰트 크기 옵션

| 설정 | ESC/POS 커맨드 | 폭(chars) | 비고 |
|------|---------------|-----------|------|
| 기본(small) | `GS ! 0x01` 1×2배높이 | 42 | 헤더·정보 라인 |
| 보통(normal) | `GS ! 0x01` 1×2배높이 | 42 | 메뉴명 기본 |
| 크게(large) | `GS ! 0x11` 2×2배 | 21 | 짧은 메뉴명 |

> **주의**: `large` 모드(21자)는 메뉴명이 21 인쇄폭 이하인 경우에만 사용.  
> "아이스 아메리카노 (상가사장님 가격)" = 35폭 → `normal`(42자)에서만 한 줄 표시 가능.
